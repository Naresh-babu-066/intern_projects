# intern_projects
Static Website – DMV Test Platform

Overview
This project is a DMV-based static website designed for a computer-based driving license test. The site is responsive and works smoothly across laptops, tablets, and mobile devices.

Home Page – Main interface with video and navigation menu

About Page – Describes the DMV system and features

Contact Page – Displays map and contact details

Tools & Technologies Used

Category	Tools / Technologies

Code Editor	Visual Studio Code (VS Code)

Version Control	Git & GitHub

Frontend Languages	HTML5, CSS3, JavaScript

Design System	CSS Grid & Flexbox

Media Elements	Video, Images, Embedded Google Maps

Browser Testing	Chrome, Edge, Firefox, Mobile Browser DevTools

Steps to Open and Test the Site

1. Clone the Repository
2. Open your terminal and run:
3. git clone https://github.com/Naresh-babu-066/intern_projects
4. Navigate to the Project Folder
5. cd intern_projects/static_site
6. Open in Visual Studio Code
7. Launch VS Code Open the folder: intern_projects/static_site
8. Run the Project
Option 1: Open Home.html directly in your browser
Option 2: Use Live Server in VS Code for auto-reload Right-click on Home.html
9. Select "Open with Live Server"
10. Test Responsiveness
Open Developer Tools (Ctrl + Shift + I → Toggle Device Toolbar)

Check how the website looks on Laptop, Tablet, and Mobile views
